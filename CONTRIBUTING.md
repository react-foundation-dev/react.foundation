# Contributing to React Foundation

Thanks for contributing to the React Foundation website.

This top-level file is intentionally short. The canonical contribution guide
lives in [`.github/CONTRIBUTING.md`](./.github/CONTRIBUTING.md). It covers
setup, branching, test-first development, pull request requirements, review
standards, and label conventions.

Maintainers use
[`.github/MAINTAINER_WORKFLOWS.md`](./.github/MAINTAINER_WORKFLOWS.md) for
issue triage and pull request review.
